开发者AI插件：高效编码与问题解决
专为开发团队设计的插件，集智能编码、问题排查和解决方案推荐于一体。提升开发效率，加速问题解决。

<div >

<h3 >Chat</h3>

[Chat] 对话框支持多种命令组合与gpt对话，快捷键 ctrl/meta L 唤起对话框。

<h3>Autocomplete</h3>

[Tab] 编辑代码停顿后自动获取补全，通过快捷键tab快速采纳。 或者通过快捷键 ctrl alt/ctrl meta  K 手动触发补全提示。

<h3>Edit</h3>

[Edit] 代码编辑可以在原生编辑器里面调起代码编辑指令，快捷键 ctrl/meta I 唤起编辑。

</div>
